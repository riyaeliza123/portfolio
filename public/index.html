Pinniped census application using YOLOv11 - Drone imagery

Sept 2025 – Dec 2025

Developed a Computer Vision model using YOLO v11 to automatically detect and count pinnipeds in drone imagery. Trained on 294 images from Vancouver Island locations (Campbell River, Nanaimo, Cowichan), expanding to 700+ images through preprocessing and augmentation. Built a user-friendly Streamlit application enabling researchers to upload drone images, visualize detections with annotated bounding boxes, and download per-image and per-location pinniped counts for analysis.

Computer Vision Object detection Streamlit Python
